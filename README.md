# WarvilPHP Framework

A lightweight PHP framework inspired by Laravel.

## Features

- MVC Architecture
- Routing System
- Database ORM
- CLI Commands
- Component-based Views
- Middleware Support
- API Response Handling

## Requirements

- PHP 8.0 or higher
- Composer
- MySQL Database

## Installation

### Option 1: Via Composer

```bash
composer create-project wardvisual/warvilphp your-project-name
